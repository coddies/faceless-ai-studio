# GitHub Collaborators Guide

To ensure the hackathon judges can review your private repository, follow these steps to add them as collaborators.

## Required Emails

Please add the following emails as collaborators:

1. `testing@devpost.com`
2. `Amazon-Nova-hackathon@amazon.com`

## Step-by-Step Instructions

1. **Open your repository** on GitHub.
2. **Click on the **Settings** tab at the top.
3. **In the left sidebar, click **Collaborators** (under the "Access" section).
4. **Click the green **Add people** button.
5. **Enter the email address** (e.g., `testing@devpost.com`) into the search box.
6. **Select the user and click **Add [user] to this repository**.
7. **Repeat for the second email address**.

> [!IMPORTANT]
> Once you add them, GitHub sends an invite. They must accept the invite to see your code. Usually, Devpost judges use scripts to accept these automatically.

## Security Reminder

Your API keys are stored in the `.env` file, which is listed in `.gitignore`. This ensures your keys are **never** uploaded to GitHub, keeping your AWS and AI accounts safe.
