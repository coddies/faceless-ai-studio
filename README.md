# 🎬 Faceless AI Studio

An all-in-one YouTube content creation tool powered by **Amazon Nova AI**. Generate scripts, cinematic scenes, thumbnails, voiceovers, and SEO titles—everything you need for a faceless channel in one place.

Built for the **Amazon Nova Hackathon 2026**. #AmazonNova

---

## 🚀 Features

- **AI Script Writer**: Generate engaging video scripts or lyrics with Nova Pro.
- **Scene Analysis & Generation**: Analyzes your script to create matching cinematic scenes using Nova Canvas.
- **Voice Generator**: Convert scripts to professional voiceovers using Amazon Polly.
- **Thumbnail Designer**: Create high-quality, style-consistent thumbnails.
- **SEO Suite**: Automatically generate optimized titles, descriptions, and tags.

---

## 🛠️ Local Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/coddies/faceless-ai-studio.git
   cd faceless-ai-studio
   ```

2. **Environment Variables**:
   Update your API keys. Copy `.env.example` to `.env` and fill in your credentials:
   ```bash
   cp .env.example .env
   ```

3. **Install Dependencies**:
   ```bash
   cd backend
   pip install -r requirements.txt
   ```

4. **Run the Server**:
   ```bash
   python main.py
   ```
   Open `http://localhost:8000` in your browser.
